# Getting Started with Denali
<Lead>If you're just getting started, this guide offers comprehensive insights to help you seamlessly integrate Denali into your React projects.</Lead>


## Prerequisites

Make sure you have the following installed:

- Node.js (version 12 or higher)
- npm or yarn
- React (version 17 or higher recommended)
- A code editor (VS Code recommended)

---

## Installation

To install Denali, run the following command in your React project's root directory:

```bash
npm install denali
```
