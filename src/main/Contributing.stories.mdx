## Getting Started

Fork the repository and clone it to your local machine. Ensure that you have Node.js and npm/yarn installed. Follow the README for the initial setup.

---

## Contribution Guidelines

Please adhere to the following principles:

- Maintain WCAG 2.1 AA and Section 508 accessibility standards.
- Ensure your components use React hooks.
- Test all components before committing.

---

## Code Contribution

1. Fork the repo and clone it locally.
2. Create a new branch for your feature or bug fix.
3. Write code following the project's coding standards.
4. Test your code thoroughly.
5. Commit and push your changes to your fork.
6. Open a pull request against the main branch.

---

## Figma Documentation

If your contribution involves design elements, please update the Figma documentation to reflect your changes. Ensure your Figma components match the React components 1:1.

---

## Pull Requests

Before submitting a pull request, please ensure:

- All tests pass.
- The code has been reviewed by at least one other person.
- You have updated the Figma documentation if applicable.
- Your commits have clear and descriptive messages.
