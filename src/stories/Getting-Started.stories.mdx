# Getting Started with Denali

---

## Introduction

Welcome to the "Getting Started" guide for Denali. If you're new here, this guide provides all the information you'll need to start using Denali effectively in your React projects.

---

## Prerequisites

Make sure you have the following installed:

- Node.js (version 12 or higher)
- npm or yarn
- React (version 17 or higher recommended)
- A code editor (VS Code recommended)

---

## Installation

To install Denali, run the following command in your React project's root directory:

```bash
npm install denali
```
